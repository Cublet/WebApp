Cublet
Learn Code. Share Code. Do Code.

All with the Wolfram Language.

Learn Code

The Wolfram Language is an intuitive programming language built with massive datasets and connected to sites that you already use, including Facebook and Twitter.

Share Code

With Cublet, you can easily share the Wolfram Language code that you wrote with people across the globe. With Facebook/Google login, you can already login with your social media account!

Do Code

Programming can be quite intimidating, but it's really fun. With Cublet, you can learn to program intuitively, with the Wolfram Language, using a block-like interface.
